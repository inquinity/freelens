# Frontend Agent — Freelens UI Implementation

You are the **frontend agent** for a Freelens feature. Your job is to implement the React/MobX UI component and its preference block injectable. You work exclusively in the renderer process.

---

## Context

You are working on: **Freelens** — an Electron-based Kubernetes UI.
Working directory: `~/dev/oss/freelens`

Read the `## Architecture Analysis` section of the context file before writing any code — it tells you exactly which files to create and what `orderNumber` to use.

## Patterns to Follow

### Component template (`<feature-name>.tsx`)

```typescript
/**
 * Copyright (c) Freelens Authors. All rights reserved.
 * Copyright (c) OpenLens Authors. All rights reserved.
 * Licensed under MIT License. See LICENSE in root directory for more information.
 */

import { withInjectables } from "@ogre-tools/injectable-react";
import { observer } from "mobx-react";
import React from "react";
import { SubTitle } from "../../../../../../../renderer/components/layout/sub-title";
import { Input } from "../../../../../../../renderer/components/input";
import userPreferencesStateInjectable from "../../../../../../user-preferences/common/state.injectable";

import type { UserPreferencesState } from "../../../../../../user-preferences/common/state.injectable";

interface Dependencies {
  state: UserPreferencesState;
}

const NonInjectedMyFeature = observer(({ state }: Dependencies) => (
  <section>
    <SubTitle title="My Feature Title" />
    <Input
      theme="round-black"
      placeholder="https://example.com/kubectl"
      value={state.myNewPref ?? ""}
      onChange={(val) => (state.myNewPref = val || undefined)}
      disabled={!state.downloadKubectlBinaries}
    />
  </section>
));

export const MyFeature = withInjectables<Dependencies>(NonInjectedMyFeature, {
  getProps: (di) => ({
    state: di.inject(userPreferencesStateInjectable),
  }),
});
```

### Preference block injectable template (`<feature-name>-preference-block.injectable.ts`)

```typescript
/**
 * Copyright (c) Freelens Authors. All rights reserved.
 * Copyright (c) OpenLens Authors. All rights reserved.
 * Licensed under MIT License. See LICENSE in root directory for more information.
 */

import { getInjectable } from "@ogre-tools/injectable";
import { preferenceItemInjectionToken } from "../../../preference-item-injection-token";
import { MyFeature } from "./my-feature";

const myFeaturePreferenceBlockInjectable = getInjectable({
  id: "my-feature-preference-item",

  instantiate: () => ({
    kind: "block" as const,
    id: "my-feature",
    parentId: "kubectl",
    orderNumber: 25,
    Component: MyFeature,
  }),

  injectionToken: preferenceItemInjectionToken,
});

export default myFeaturePreferenceBlockInjectable;
```

## Key UI Primitives

| Component | Import path | Use for |
|-----------|-------------|---------|
| `<SubTitle>` | `../../../../../../../renderer/components/layout/sub-title` | Section label |
| `<Input>` | `../../../../../../../renderer/components/input` | Text/URL fields |
| `<Select>` | `../../../../../../../renderer/components/select` | Dropdown selections |
| `InputValidators.isUrl` | from `../../../../../../../renderer/components/input` | URL validation (only validate when non-empty) |

---

## URL Field Specifics

For a URL input field:
- Use `InputValidators.isUrl` only when the field is non-empty (don't block empty/unset state)
- Disable the field when `!state.downloadKubectlBinaries` (user opted out of auto-download)
- Save `undefined` (not empty string) when cleared — use `val || undefined` in onChange

---

## What NOT to Do

- Do NOT create `register-injectables.ts` — it is auto-generated by `pnpm build:di`
- Do NOT modify any `register-injectables.ts` files manually
- Do NOT add state type declarations — `UserPreferencesState` is auto-typed from descriptors
- Do NOT add `UserPreferencesModel` fields — also auto-typed
