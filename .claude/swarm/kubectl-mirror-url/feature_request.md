# Feature Request: Custom kubectl download mirror URL

**GitHub Issue:** https://github.com/freelensapp/freelens/issues/1823
**Status:** Open | **Label:** enhancement | **Author:** @inquinity

---

**What would you like to be added?**

Add an optional preference setting to specify a custom base URL for downloading kubectl binaries (in addition to the existing Default (Google) and China (Azure) mirror options).

**Why is this needed?**

Many organizations run in environments where direct Internet access is restricted by firewall policy. This will allow Freelens to download the correct kubectl binaries in restricted environments.
